About Me

Hey, my name is Martin Saad. The idea for this website came up a few months ago. There was a book I wanted to read but didn't have it, so I tried to see if any one of my friends might have it. After some searching with no result, I tried to see if there are some fb groups to swap books. I mean, I have some books I don't read anymore, so I guess my desired book might be sitting in someone's shelf and we could exchange. Eventually a friend gave my the book but it was harsh.

The perpuse of bookswap is to allow people to exchange and swap books for FREE. I believe there is no reason why we shouldn't help each other with books we already read and they are just sitting in the shelf.

If you have any ideas to add or improve the website don't hasitate to share. Also, the website is an MVP and it's not perfect :) If you see any bugs, please share: martin.saad.ms@gmail.com
